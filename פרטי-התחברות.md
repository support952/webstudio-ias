# פרטי האתר והתחברות — Web Studio IAS

## כתובות (URLs)

| סביבה   | כתובת |
|---------|--------|
| פיתוח   | http://localhost:5000 |
| פאנל ניהול | http://localhost:5000/ws-panel-9x7k |

## התחברות לפאנל הניהול

- **כתובת:** http://localhost:5000/ws-panel-9x7k  
- **אימייל:** הערך של `ADMIN_EMAIL` בקובץ `.env` (למשל `admin@webstudio.local`)  
- **סיסמה:** הערך של `ADMIN_PASSWORD` בקובץ `.env`  

אלה פרטי ההתחברות האמיתיים של הפאנל — יש להגדיר אותם ב־`.env` ולהפעיל מחדש את השרת כדי שהמשתמש ייווצר.

## הרצת הפרויקט

```powershell
npm run dev
```

אחרי שהשרת עולה, לגלוש ל־http://localhost:5000 או לפאנל הניהול.

## משתני סביבה (`.env`)

ההגדרות נמצאות בקובץ `.env` בשורש הפרויקט:

- `DATABASE_URL` — חיבור ל־PostgreSQL  
- `ADMIN_USERNAME` — שם תצוגה לאדמין  
- `ADMIN_EMAIL` — אימייל להתחברות לפאנל הניהול  
- `ADMIN_PASSWORD` — סיסמה להתחברות לפאנל הניהול  

**חשוב:** הקובץ `.env` לא עולה ל־Git; הסיסמאות נשארות רק אצלך במחשב.
